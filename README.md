# RelearnThreeJS

## 介绍

重新系统性的学习 threejs, 做一个备忘~

## 运行

- 安装依赖
    npm install
    
- 开启本地服务
    npm start  // 启动在 9010端口
    
- 根绝 src/client/ 的路径在浏览器中访问即可，例如 src/client/scene/demo1.html  -> http://127.0.0.1:9010/scene/demo1.html
